# Jekyll Bootstrap

A Jekyll Bootstrap with i18n, collections, YML data, Google Analytics and other features.

## Development

Dependencies:

- ruby v1.9
- jekyll v2.5.2

To start development, type:

    $ jekyll serve --watch

Open your browser at [http://localhost:9000/](http://localhost:9000/)

## Deployment

Dependencies:

- node v0.10
- grunt-cli v0.1


    $ npm i -g grunt-cli
    $ npm i

To publish, type:

    $ grunt
